# RCord GameMaker Client (GML)

Этот каталог — заготовка GameMaker-проекта. Здесь описана структура сцен и
выложены скрипты GML для сетевого протокола и хранения состояния. Проект можно
подключить к полноценному `.yyp` позже, сохраняя эту структуру.

## Сцены (rooms)

- `rm_login`
  - Экран входа/регистрации.
  - Поля логина/пароля, кнопки **Login**/**Register**.
  - После `register` автоматически переходить на `rm_main`.
  - После `login` загружать списки комнат, чатов и пользователей.

- `rm_main`
  - Три зоны: **1/5** (левый блок), **3/5** (центр), **1/5** (правый блок).
  - Каждая зона имеет рамку 1px, цвет `#3b3d45`.
  - Тёмная тема Discord-стиля (см. `ui/theme.gml`).

## Блоки UI

### Левый блок (1/5)
- Список комнат и чатов.
- Кнопка `+` для создания комнат.
- При выборе канала скрывать остальные чаты до выхода через кнопку
  **Список каналов**.

### Центральный блок (3/5)
- Текстовый чат: сообщения, файлы, изображения, эмодзи.
- Голосовой чат:
  - авто-включение микрофона,
  - выбор устройства ввода,
  - индикация активности 2px зелёной рамкой вокруг аватара,
  - демонстрация экрана.

### Правый блок (1/5)
- Список участников.
- Индикатор статуса: круг 4px (`#23a55a` online, `#4f545c` offline).
- Подсветка никнейма при online.

## Настройки клиента
- Форма IP/порта сервера (+ media-порт при необходимости).
- Сохранение/загрузка настроек — `scripts/settings.gml`.

## Сетевой протокол
- Newline-delimited JSON.
- Реализовано в `scripts/net_protocol.gml`.

## Хранение данных
- Клиент всегда полагается на сервер (`DB.dat`).
- Локально хранится только конфиг подключения.
